## 2026-06-22 - Module 2 成本追踪与预算控制

### 已实现

- 新增 `src-tauri/src/costs` 成本模块。
- 实现成本分类: LLM、Embedding、Reranker、Search API、HTTP Skill、Approval、Other。
- 实现 `NewCostRecord`、`CostRecord`、`CostQuery`、`TokenUsage`。
- 实现 token 成本估算，支持从 Module 1 的 `ModelConfig` 读取每百万 token 价格。
- 实现预算决策:
  单次预算、每日预算、月度预算、普通模式高成本确认、Auto Mode 预授权预算。

- 实现 SQLite 成本账本 `SqliteCostLedger`，默认数据库文件为 `costs.db`。
- 实现成本数据库幂等迁移和 `schema_migrations` 记录。
- 成本记录支持 provider、model、workflow、run、node、tool_call_id、tokens 和 metadata。
